# wxc-popup-tag

> 弹出层选择tag - 小程序组件

## Install

``` bash
$ min install @minui/wxc-popup-tag
```


## API

### PopupTag

| 名称                  | 描述                         |
|----------------------|------------------------------|
|`prop-name`           | 描述属性的类型，默认值等         |
|`method-name`         | 描述方法的参数，返回值等         |
